\ssc{Q1 Part (c)}{

We will test if \uw{\bth}{1} and \uw{\bth}{2} are statistically significant (i.e. testing \uw{H}{0} : \uw{\bth}{1} \eql 0 and \uw{\bth}{2} \eql 0)

Our Wald statistics (given by the z value in the summary) are:

\uw{\bth}{1} : 1.16

\uw{\bth}{2} : 1.87

We will test at \afa \eql 0.05

\uw{Z}{\afa / 2} \eql \uw{Z}{0.05 / 2} \eql  \uw{Z}{0.025} \eql $-$1.96

\uw{\bth}{1} : \av{1.16} \gr 1.96 \lra Not true.

\uw{\bth}{2} : \av{1.87} \gr 1.96 \lra Not true.

Thus, 

\textbf{we fail to reject \uw{H}{0}.}

\begin{verbatim}
new <- data.frame(x1 = 45000, x2 = 5)
predict(glm.1, new)
\end{verbatim}

gives us 1.214124

So,

\eqn{\ln(p / (1 - p)) = 1.214124}
\eqn{p / (1 - p) = e^{1.214124}}
\eqn{p = e^{1.214124}(1 - p)}
\eqn{p = e^{1.214124} - pe^{1.214124}}
\eqn{(1 + e^{1.214124})p = e^{1.214124}}
\eqn{p = \frac{e^{1.214124}}{(1 + e^{1.214124})}}
\eqn{p(y = 1 \textrm{ } | \textrm{ } x1 = 45,000, \textrm{ } x2 = 5) = 0.77102783}

\textbf{They have a 77.1\% chance of buying a new vehicle in the next 6 months.}
