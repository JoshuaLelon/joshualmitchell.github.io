\bsc{Theorem 6.1.10: The Chain Rule}{

Recall:

f(x) \eql \uf{x}{n} implies f\pr(x) \eql n\uf{x}{n - 1} \fa n \mem \bnt{\bz}{\bk{0}}

\lt{I, J be intervals in \br and f : I \lra \br and g : J \lra \br, where f(I) \sbs J}

Also,

\lt{c \mem I}

If f\pr(c) exists and g\pr(f(c)) exists, then (g o f)\pr(c) \eql g\pr(f(c)) * f\pr(c)

\bgpf 

Since g\pr(f(c)) exists,
\eqn{\limt{y}{f(c)} \frac{g(y) - g(f(c))}{y - f(c)} = g\pr(f(c))}

\

Define the function h : J \lra \br where
\eqn{h(y) = \frac{g(y) - g(f(c))}{y - f(c)}}
if y $\neq$ f(c), and
\eqn{h(y) = g\pr(f(c))}
if y \eql f(c).

Notice that \limt{y}{f(c)} h(y) \eql g\pr(f(c)) \eql h(f(c)) \bpth{1}

Since f(c) \mem J, which is an interval, f(c) is a limit/accumulation point of J.

Recall: I is an interval if where ever \uw{x}{1}, \uw{x}{2} \mem I, \uw{x}{1} \ls \uw{x}{2}, and \uw{x}{1} \ls x \ls \uw{x}{2} then x \mem I

Then, by Theorem 5.2.2(a)(d),

\bpth{a} h is continuous at f(c)

\bpth{d} \limt{y}{f(c)} h(y) \eql h(f(c))

h is continuous at f(c).

Also, since f is differentiable at c, it follows by Theorem 6.1.6 that f is continuous at c.

Also, h is continuous at f(c).

By Theorem 5.2.12, h o f is continuous at c.

By Theorem 5.2.2 (a)(d), we see that

\eqn{\limt{x}{c} (h \textrm{ o } f)(x) = \limt{x}{c} \textrm{ } h(f(x)) = h(f(c)) = g\pr(f(c)) \textrm{ from (1)}}

Now, let x \mem I with x $\neq$ c.

Then,

\step{(g \textrm{ o } f)(c) \eql \limt{x}{c} \frac{(g  \textrm{ o } f)(x) - (g \textrm{ o } f)(c)}{x - c} = \limt{x}{c} \frac{g(f(x)) - g(f(c))}{x - c}}{2}

Notice that h(y)(y \ms f(c)) \eql g(y) \ms g(f(c)), \fa y \mem J.

Thus,

\step{h(f(x))[f(x) - f(c)] = g(f(x)) - g(f(c))}{3}

Substituting \bpth{3} into \bpth{2} yields that
\eqn{(g \textrm{ o } f)\pr(c) = \limt{x}{c} \textrm{ } \frac{h(f(x))[f(x) - f(c)]}{x - c} = \limt{x}{c} [h(f(x))] \limt{x}{c}[\frac{f(x) - f(c)}{x - c}] = g\pr(f(c)) * f\pr(c)}

Hence, result.

You might be wondering, though, why can't you do this?

\eqn{(g \textrm{ o } f)\pr(c) = \limt{x}{c} [\frac{g(f(x)) - g(f(c))}{x - c} * \frac{f(x) - f(c)}{f(x) - f(c)}] = \limt{x}{c} [ \frac{g(f(x)) - g(f(c))}{f(x) - f(c)} ] * \limt{x}{c} [\frac{f(x) - f(c)}{x - c}]}

You can't let f(x) \ms f(c) \eql 0. It might be 0.
