\ssc{Exercise 2}{

Which of the following binary operations are associative?
\balist
\item subtraction of integers
\item division of nonzero rationals
\item function composition of polynomials with real coefficients
\item multiplication of 2 x 2 matrices with integer entries
\item exponentiation of integers
\elist

}

\ssc{Exercise 4}{
Which of the following sets are closed under the given operation?
\balist
\item {0, 4, 8, 12} addition mod 16
\item {0, 4, 8, 12} addition mod 15
\item {1, 4, 7, 13} multiplication mod 15
\item {1, 4, 5, 7} multiplication mod 9
\elist

}

\ssc{Exercise 23}{

(Law of Exponents for Abelian Groups)

Let a and b be elements of an Abelian group and let n be any integer. Show that \uf{(ab)}{n} \eql \uf{a}{n}\uf{b}{n}.

Is this also true for non-Abelian groups?

}

\ssc{Exercise 25}{

Prove that a group G is Abelian iff \uf{(ab)}{-1} \eql \uf{a}{-1}\uf{b}{-1}, \fa a, b \mem G.

}
