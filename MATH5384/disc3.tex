\ssc{Exercise 58}{
\textbf{Let S be the set of real numbers. If a, b \mem S, define a \rln b if a \ms b is an integer.}

\balist
\item \textbf{Show that \rln is an equivalence relation on S.}

Properties of an equivalence relation:

Reflexive: \fa a \mem S, a \rln a

Symmetric: a \rln b \rar b \rln a

Transitive: a \rln b and b \rln c \rar a \rln c

\bgpf 

Let a \mem S

a \mem \br \rar a \eql a.

Therefore, a \ms a \eql 0 \mem \bz

Hence, (a, a) is a member of the relation \fa a \mem S.

Thus, \rln is a reflexive relation on S.

\

Let a, b \mem S such that a \ms b \eql c where c \mem \bz

a \ms b \eql c

a \eql c \ps b

a \ms c \eql b

$-$c \eql b \ms a

Notice that c \mem \bz \rar $-$c \mem \bz

Thus, if a \ms b yields an integer, then b \ms a yields an integer.

Hence, \rln is a symmetric relation on S.

\

Let a, b, c \mem S such that a \rln b and b \rln c.

Thus, \exs d, e \mem \bz such that a \ms b \eql d and b \ms c \eql e.

Notice that d \ps e \eql a \ms b \ps b \ms c \eql a \ms c

Since d, e \mem \bz \rar (d \ps e) \mem \bz, a \ms c yields an integer.

Hence, \rln is a transitive relation on S, and that completes the proof.

\epf 

\item \textbf{Describe the equivalence classes of S.}

Given a, b \mem S, a \rln b if a \ms b \eql c where c \mem \bz

So each equivalence class is a set of real numbers each separated by some integer.
